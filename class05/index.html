there are three pseudo classes that allow you to changed the appearance of elements when a usuer is interacting with them

  • Hover: applied when a user hovers over an element with a pointing device like a mouse. this is commonly used to change the appearance of links and buttons when a user places the cursor over them
  • Active: applied when a n element is being activated for a user; when a button is being pressed or a link being clicked. it makes the button or link feel like it s actually being pressed
  • Focus: applied when an element has focus, any element that you can interact with like a link you can click on in or any form control can have focus
