    CSS or Cascading style sheets

    To simplify what CSS does, it is a design tool that allows you to modify your html document. You can edit the text font and the color, the background of the webpage, decide where the text appears on the screen so on and so forth. The key to mastering css is by first understanding the basic properties and then learning how to apply them properly. The most important thing is practice.

    One thing I have had trouble with is the linking of your html and CSS. To do that you have to have an href, the type, and the rel.

    • href: specifies the path to the css file example: href="css/styles.css"
    • type: this specifies the type of document being linked to. the value should be text/css example: type="text/css"
    • rel: specifies the relationship between the html page and the file it is linked to. value is stylesheet when linked to css example: rel="stylesheet"
