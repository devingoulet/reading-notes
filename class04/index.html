Links

Links are the defining feature of the web becasue they allow you to move from one web page to another - thus the term surfing the web.

  • Links are created using the a element, you specidy which page you want to link to using the href attribute
  • The text between the opening and close a tag is known as the link text
  • the link text should explain where the visitors will be taken if they click the link
  • when you link to a different website the value of the href attribute will be the full web address of the site, known as the absolute URL

positioning schemes

positioning schemes that allow you to control the layout of a page: normal flow, relative positioning, and absolute positioning. you specify the positioning scheme using the position property in css. You can also float elements using the float property

  • normal flow: specifying the width of the boxes they will not appear next to eachother becuase this is their default behavior.
  • relative positioning: this moves an element from the position it would be in normal flow, shifting it to the top,right,bottom, or left of where it wouold have been placed. this does not effect the position of the surrounding elements
  • absolute positioning: this positions the element in relation to its containing element. it is taken out of the normal flow, meaning that it doesnt effect the position of any surrounding elements.
  • fixed positiong: this is a form of absolute positioning that positions the elements in relation to the browser window as opposed to the containing element.
  • floating elements: an element allows you to take that element out of normal flow and position it to the far left or right of a containing box.

Javascript Functions

functions let you group a series of statements together to perform a specific tasl. if different parts of a script repeat the same task, you can reuse the function

some functions need to be provided with information in order to achieve a given task. Example, a function to calculate the are of a box would need to know its height and width. Pieces of info that are passed through functions are called parameters.

These statements update the message at the top of the page. The function acts like a store; it holds the statements that are contained in the curly braces until you are ready to use them. Those statements are nut run until the function is called.

To create a function, you give it a name and then write the statements needed to achieve its task inside the curly braces. This is the function declaration.

  • you declare a function, using the function keyword
  • you give the function a name/identifier
  • the statement that preforms the task will be inside curly braces
  • if a function needs specific information to run, when its declared you give the function parameters
  • if a function needs info you indicate what it needs to know inside parantheses, those items inside are the parameters

Getting values out of functions

  • some functions return info to the code that called them, when performing a calculation theyll return the result
  • functions can return more than one value when using an array
