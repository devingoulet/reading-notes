Loops check a condition. if it returns true, a code black will run. then the condition will be checked again and if it still returns true, the code block will run again. it repeats until the condition returns false.

  • For loop: used to run code a specific number of times. in a for loop, the condition is usually a counter which is used to tell how many times the loop should run
  • While loop: used for when you dont know how many times the code should run. the condition can be something other than a counter, and the code will continue to loop for as long as the condition is true
